Pages

Snippets can be organized into pages.

Page structure

A page is a simple html file. Pages can contain any HTML content and include snippets.
Pages can contain content like text and explanations. Is it important to note that the same snippet can be included by multiple pages, so if something is worth sharing throughout pages, it is worth to create a snippet for it. Examples:
  • pages/
    • page1.html
    • page2.html
    • page3.html

Special tags

The tag 
<snippet>id</snippet>
lets you include a snippet in the page.
