Floating snippets (work in progress)

It is possible to create links to snippets that will display the snippet when the mouse is over them.

HTML

If you're writing HTML for a page or a snippet, you can embed a floating snippet link as follows 
    <ref snippet="snippet-id">link text</ref>

LaTeX

If you're writing a snippet using LaTex, you can embed a floating snippet link as follows 
    \snippetref{snippet-name}{link text}
