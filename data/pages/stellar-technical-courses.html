Courses

Pages can be organized into courses.

Course structure

A course is a simple json file. Courses contain a list of pages and their title level. Examples:
  • courses/
    • course1.html
    • course2.html
    • course3.html

Example

    {
        "title": "Technical Guide",
        "pages": [
            [
                1,
                "Introduction",
                "stellar-technical-introduction"
            ],
            [
                1,
                "Content structure",
                "stellar-technical-content-structure"
            ],
            [
                2,
                "Snippets",
                "stellar-technical-snippets"
            ],
            [
                2,
                "Pages",
                "stellar-technical-pages"
            ],
            [
                2,
                "Courses",
                "stellar-technical-courses"
            ]
        ]
    }
