API

POST /course/<id>

Returns the course JSON file.

POST /universe/<id>

Returns the universe JSON file.

POST /page/<id>

Returns the course HTML file.

POST /snippet/<id>

Returns the primary file in the snippet folder, which is in data/snippets/<id>.

GET /snippet/<id>/<file>

The snippet folder may contain other files that will be requested by the main file. These files can be accessed using this route.

GET /course/<id>

Returns the course page.

GET /universe/<id>

Returns the universe page.

GET /page/<id>

Returns the page page.

GET /snippet/<id>

Returns the snippet page.

GET /search

Returns the search page.

GET /<file>

Returns a static file.

GET /

Returns the index file.

POST /query/snippet/<keyword>

Returns a JSON lookup result for the matching snippets.

POST /query/page/<keyword>

Returns a JSON lookup result for the matching pages.

POST /query/courses/<keyword>

Returns a JSON lookup result for the matching courses.

POST /query/universe/<keyword>

Returns a JSON lookup result for the matching universe.
