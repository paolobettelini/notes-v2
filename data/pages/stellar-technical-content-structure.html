Data folder

Every piece of content of the website is put in a folder named data.

The content is split into 3 categories
  • Snippets
  • Pages
  • Courses
For each of these categories, a folder is present in data/.
  • data/
    • snippets/
    • pages/
    • courses/
Every snippet, page or course is identified by a string identifier.
For example, the id for this course is stellar-technical-guide (see URL) and the id for this page is stellar-technical-content-structure.
