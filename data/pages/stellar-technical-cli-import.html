Importing data to MongoDB

The stellar CLI tool can be used to initialize data to the MongoDB database. 
    stellar-cli import -c "..." --import ./data
    stellar-cli import -c "..." --import ./data/snippets
    stellar-cli import -c "..." --import ./data/pages
    stellar-cli import -c "..." --import ./data/courses
    stellar-cli import -c "..." --import ./data/snippets/snippet1
    stellar-cli import -c "..." --import ./data/pages/page1.html
    stellar-cli import -c "..." --import ./data/courses/course1.json
    stellar-cli import -c "..." --import course1.json course2.json
Example of connection URL: mongodb://127.0.0.1:27017
